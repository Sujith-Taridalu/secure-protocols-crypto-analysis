# Secure Protocols & Crypto Analysis

## Overview

This repository consolidates practical security analysis work across cryptography, certificates, PKI, network protocols, and transport security concepts. Rather than presenting isolated assignments, it is positioned as a security foundations lab focused on explaining how secure systems work and where they fail.

## Objectives

- document practical understanding of encryption and authentication concepts
- analyze certificate trust, protocol behavior, and secure communication models
- explain protocol security properties and implementation tradeoffs
- connect theory to operational security reasoning

## Project Highlights

- Reviewed RSA, modular arithmetic, encoding/decoding, and cipher concepts
- Analyzed certificates, certificate chains, and CA trust
- Studied block cipher modes such as ECB and CBC
- Covered IPSec anti-replay protections and secure packet handling
- Evaluated protocol and crypto questions involving Double-DES, MACs, and non-repudiation
- Interpreted traceroute/network path behavior from a cyber range environment

## Topic Areas

- PKI and digital certificates
- certificate chain validation
- RSA and asymmetric cryptography
- cipher modes and message authentication
- IPSec and replay protection
- protocol-level trust and implementation risk
- network path interpretation and secure communication flow

## Skills Demonstrated

`Cryptography Fundamentals` `PKI` `TLS/Certificates` `IPSec` `Network Security` `Protocol Analysis` `Risk Explanation` `Technical Documentation`
