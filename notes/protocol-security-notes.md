# Protocol Security Notes

This repository groups together practical observations across encryption, authentication, certificate trust, and secure network behavior. The underlying work shows a broad foundation in how secure communication mechanisms are supposed to work and what tradeoffs they introduce.

## Key Areas Covered

- RSA and asymmetric encryption concepts
- digital certificates and chain-of-trust reasoning
- block cipher mode differences such as ECB versus CBC
- MAC limitations and non-repudiation discussion
- IPSec replay protection and packet validation
- protocol visibility and path interpretation through network tools

## Intended Audience

This material is best presented for recruiters or engineers who want to see solid security fundamentals and good written reasoning, even when the project is more analytical than code-heavy.
